## APEX下拉式選單

### Apex official website

點選` Apex official website `會進入apex的官方網站

### Legends

點選` Legends `會進入` legends.html `的頁面
<br>
點選下方各個圖片會進入各個英雄的介紹頁面

### Season Videos

點選` Season Videos `會進入` seasons.html `的頁面
<br>
點選下方各個圖片會進入各個賽季的` YOUTUBE `介紹影片
